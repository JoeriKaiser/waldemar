Invar is a terminal task manager built with Go and [Charmbracelet](https://charm.sh) libraries (Bubble Tea, Lipgloss). It provides an elegant TUI for managing tasks directly from the terminal.

Features include task creation, editing, completion tracking, and filtering - all with beautiful terminal UI components and keyboard-driven navigation.
